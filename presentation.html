Title · 標題

EN

SweetHeart — Smart Grid Typography Studio

A browser-based layout lab for visual communication: grid systems, typographic presets, rulers, and non-destructive alignment guides—so designers can iterate quickly before final production in print or motion.

中文

SweetHeart — 智慧網格排版工作室

以瀏覽器運行的版面實驗工具：整合多種網格系統、排版預設、像素標尺與參考線，協助視覺傳達設計在定稿前快速試排與對齊。

Why we built it · 為什麼做這個工具

EN

Students and junior designers often jump between reference PDFs, Illustrator, and sketches. We wanted a lightweight, zero-install surface that foregrounds grid logic and typographic rhythm—the “grammar” of layout—without the full complexity of a DTP suite.

中文

在學習階段，同學常在參考文件、向量軟體與手繪之間切換。我們希望提供一個免安裝、可分享連結的環境，先把網格邏輯與字級節奏（版面的「文法」）講清楚，再進入完整排版流程。

Use in visual communication · 在視覺傳達流程中的位置

EN
  • Concept & moodboard → import key imagery as reference.
  • Grid & hierarchy → pick a preset (thirds, modular, symmetry…), align headline/body.
  • Client-ready mock → export PNG/JPG; JSON export for documentation.
中文
  • 概念與情緒板 → 匯入主視覺圖片作為參考範圍。
  • 網格與層級 → 選擇預設（三分、模組、對稱…），對齊標題與內文。
  • 提案稿輸出 → 匯出 PNG／JPG；JSON 方便附在報告或版本紀錄。

Design thinking · 設計思考

EN

We framed the problem as reducing cognitive load: one canvas, one reference image, presets that encode editorial intent. Undo/history supports exploration; bilingual UI respects both studio and classroom contexts.

中文

我們將問題界定為降低認知負荷：單一畫布、單一參考圖、以預設封裝編輯意圖。歷史紀錄支援試錯；中英切換符合工作室與課堂雙語情境。

Key UI/UX decisions · 介面與體驗決策

EN
  • Dark UI: reduces glare for long sessions; matches creative-tool expectations.
  • Rulers + subtle guides: Illustrator-like mint/gray guides; toggles avoid clutter.
  • Preset cards: show font/size/RGB at a glance—transparent “recipe” for the layout.
  • Language switch: one control in the header; rebuilds preset copy without reload.
中文
  • 深色介面：長時間操作較不累眼，並與常見創作工具心智模型一致。
  • 標尺＋淡色參考線：類 AI 參考線；可關閉避免干擾構圖判斷。
  • 預設卡片：並列字體／尺寸／色值，讓「版型配方」一目了然。
  • 語言切換：集中於頂欄；切換後重繪預設文案，無需重新載入頁面。

Development (brief) · 開發（簡述）

EN

Built as a single-page app with Fabric.js on canvas, Tailwind for layout, and client-side state for undo. No API keys are required for core features.

中文

以 Fabric.js 畫布、Tailwind 排版、前端狀態管理撤回。核心功能不需 API Key。
